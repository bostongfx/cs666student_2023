# cs666student

Keywords: Signal Processing, Filtered and Unfiltered Signals, Biosppy, Neurokit, Electrocardiogram

This assignment and its programming has been focused on ECG data that involves Theoretical concepts of Signal processing and its application in ECG(Electrocardiogram)
to read the heart signals and interpret them in a logical and representative manner.
This projected inculcated the use of Python programming language along with its open source libraries mainly, BIOSPPY and NEUROKIT,  numpy, matplotlib for mathematical array implementation and graphical plotting.
